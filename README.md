# API de Gerenciamento de Usuários

API REST desenvolvida para gerenciamento de usuários.

## Tecnologias

- Java
- Spring Boot
- PostgreSQL
- JPA / Hibernate
- Postman
- Git / GitHub

## Funcionalidades

- Criar usuários
- Listar usuários
- Atualizar usuários
- Deletar usuários

## Endpoints da API

GET /usuarios -> listar usuários

POST /usuarios -> criar usuário

PUT /usuarios/{id} -> atualizar usuário

DELETE /usuarios/{id} -> deletar usuário

## Testes

Os testes da API foram realizados utilizando o Postman.
